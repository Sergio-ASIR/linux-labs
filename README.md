# Linux Labs

Repositorio dedicado a laboratorios de administración Linux desarrollados durante mi formación en Administración de Sistemas Informáticos en Red (ASIR).

El contenido está organizado por áreas de administración de sistemas y documenta procedimientos prácticos relacionados con gestión de usuarios, almacenamiento y recuperación de sistemas.

---

## Contenido del repositorio

### Gestión de usuarios y grupos

Administración básica de usuarios y grupos en Linux:

* Creación y modificación de usuarios
* Gestión de grupos
* Asignación de usuarios a grupos
* Comandos básicos de administración

📁 `gestion-usuarios-grupos`

---

### Almacenamiento

Administración de discos y sistemas de archivos:

#### Particionado y sistemas de archivos

* Tabla de particiones MBR
* Particiones primarias, extendidas y lógicas
* Formateo en ext4
* Montaje de sistemas de archivos
* Verificación del almacenamiento

📁 `almacenamiento/particionado-y-sistemas-de-archivos`

#### Swap y montaje persistente

* Creación y configuración de espacio swap
* Identificación mediante UUID
* Configuración de `/etc/fstab`
* Montaje persistente tras reinicio

📁 `almacenamiento/swap-y-fstab`

---

### Backup y recuperación

Procedimientos básicos de respaldo y restauración:

* Creación de imágenes de disco
* Restauración de sistemas
* Verificación de copias de seguridad con Clonezilla

📁 `backup-y-recuperacion/clonezilla`

---

## Entorno de trabajo

* Linux
* Ubuntu Server
* VirtualBox
* Clonezilla

---

## Objetivo

Este repositorio forma parte de mi portfolio técnico y recopila laboratorios prácticos relacionados con la administración de sistemas Linux, documentando tanto los procedimientos realizados como las evidencias obtenidas durante su ejecución.
