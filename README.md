# Linux Labs

En este repositorio recopilo algunas de las prácticas de administración de sistemas Linux que he realizado durante mi formación en Administración de Sistemas Informáticos en Red (ASIR).

He seleccionado aquellas actividades que considero más representativas de los conocimientos adquiridos en gestión de usuarios, almacenamiento y recuperación de sistemas.

## Contenido

### [01 - Particionado y RAID](./01%20-%20Particionado%20y%20RAID.pdf)

Práctica centrada en la configuración de almacenamiento en Linux mediante particionado de discos, sistemas de archivos, espacio swap y configuración de RAID.

### [02 - Backup y recuperación con Clonezilla](./02%20-%20Backup%20y%20recuperación%20con%20Clonezilla.pdf)

Práctica orientada a la creación y restauración de imágenes de disco utilizando Clonezilla como herramienta de respaldo y recuperación.

### [03 - Gestión de usuarios y grupos en Linux](./03%20-%20Gestión%20de%20usuarios%20y%20grupos%20en%20Linux.pdf)

Práctica de administración básica de usuarios y grupos desde línea de comandos, incluyendo creación, modificación y asignación de permisos.

## Entorno utilizado

* Linux
* Ubuntu Server
* VirtualBox
* Clonezilla

## Sobre este repositorio

Mi objetivo es utilizar este espacio para mostrar de forma sencilla algunas de las prácticas que mejor representan mi aprendizaje en administración de sistemas Linux durante el primer curso de ASIR.
